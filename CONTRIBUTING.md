# Contributing to Backend App Tec2Med Summit

Thank you for your interest in contributing! To maintain the quality and integrity of the official version, please follow the guidelines below.

## Guidelines

1. All pull requests must undergo review before merging.
2. At least two maintainers must approve each pull request before it is merged into the main branch.
3. All commits should pass automated tests (if applicable) and follow the project's commit message conventions.
4. Add SPDX headers to all new or modified source files, using the following template:
